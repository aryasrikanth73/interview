HTML Questions

What is the difference between <div> and <span> elements in HTML?
Answer: <div> is block-level, <span> is inline.
How can you include an external CSS file in an HTML document?
Answer: Use the <link> tag in the <head> section.
What are HTML semantic elements, and give two examples?
Answer: Semantic elements describe their meaning. Examples: <header>, <footer>.

CSS Questions

What is the CSS Box Model, and what are its components?
Answer: Margin, border, padding, and content.
How do you center a block element horizontally in CSS?
Answer: Set margin: auto and specify a width.
What is the purpose of the z-index property in CSS?
Answer: Controls the stacking order of overlapping elements.

Bootstrap Questions

How do you create a responsive grid layout using Bootstrap?
Answer: Use .container, .row, and .col-* classes.
What are Bootstrap utility classes, and give an example of how to use one?
Answer: Utility classes provide quick styling. Example: .mt-3 for margin.
How do you implement a Bootstrap modal?
Answer: Use the modal structure and trigger it with a button.

PHP Questions

How do you connect to a MySQL database using PHP?
Answer: Use mysqli or PDO.
What is the purpose of the $_SESSION superglobal in PHP?
Answer: Holds session variables across pages.
How can you send an email using PHP?
Answer: Use the mail() function.

WordPress Questions

What is a WordPress shortcode, and how do you create one?
Answer: A tag that embeds content. Use add_shortcode().
How can you enqueue scripts and styles in WordPress?
Answer: Use wp_enqueue_script() and wp_enqueue_style().
What is the purpose of WordPress hooks, and what are the two types?
Answer: Allow modifying functionality. Types: actions and filters.
